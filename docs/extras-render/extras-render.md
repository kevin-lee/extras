## Get `extras-render`

* The `core` module:
  ```scala
  "io.kevinlee" %% "extras-render" % "@VERSION@"
  ```

* For `refined` library:
  ```scala
  "io.kevinlee" %% "extras-render-refined" % "@VERSION@"
  ```

or for `Scala.js`:

* The `core` module:
  ```scala
  "io.kevinlee" %%% "extras-render" % "@VERSION@"
  ```
  
* For `refined`:
  ```scala
  "io.kevinlee" %%% "extras-render-refined" % "@VERSION@"
  ```
