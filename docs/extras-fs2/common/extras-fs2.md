## Get `extras-fs2`

### For Fs2 v2
```scala
"io.kevinlee" %% "extras-fs2-v2-text" % "@VERSION@"
```
or for `Scala.js`:
```scala
"io.kevinlee" %%% "extras-fs2-v2-text" % "@VERSION@"
```

### For Fs2 v3
```scala
"io.kevinlee" %% "extras-fs2-v3-text" % "@VERSION@"
```
or for `Scala.js`:
```scala
"io.kevinlee" %%% "extras-fs2-v3-text" % "@VERSION@"
```
