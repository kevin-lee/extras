## Get `extras-core`

* The `core` module:
  ```scala
  "io.kevinlee" %% "extras-core" % "@VERSION@"
  ```

or for `Scala.js`:

* The `core` module:
  ```scala
  "io.kevinlee" %%% "extras-core" % "@VERSION@"
  ```
