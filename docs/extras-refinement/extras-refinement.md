## Get `extras-refinement`

`extras-refinement` provides syntax to use [newtype](https://github.com/estatico/scala-newtype) and [refined](https://github.com/fthomas/refined).

```scala
"io.kevinlee" %% "extras-refinement" % "@VERSION@"
```
